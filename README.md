# GenAI LLM RAG Document Q&A System

A Generative AI project that implements a Retrieval-Augmented Generation (RAG)
pipeline for document-based question answering using Large Language Models.

##  Overview
This project demonstrates how RAG enhances LLM responses by retrieving
relevant context from documents before generating answers, reducing hallucinations
and improving accuracy.

##  Tech Stack
- Python
- Large Language Models (LLMs)
- LangChain
- FAISS (Vector Database)
- Streamlit
- OpenAI-compatible APIs

##  Features
- Upload and process documents (PDF/Text)
- Generate vector embeddings
- Store embeddings using FAISS
- Retrieve relevant document chunks
- Answer user queries using RAG pipeline
